# Screenshot Guide

Use this folder for GitHub, marketplace, and documentation screenshots.

## Recommended Captures

- `focus-mode.png`: `/chat focus` confirmation and quieter chat behavior.
- `category-status.png`: `/chat status` showing allowed categories.
- `admin-notification.png`: a categorized teleport or server-notice message.
- `broadcast.png`: a purchase or announcement broadcast.

## Capture Notes

- Use a readable Minecraft GUI scale.
- Show a normal player workflow, not only admin commands.
- Redact private player messages, staff-only details, and private server names if needed.
- Keep images at least 1280px wide when possible.
